# 3312. Sorted GCD Pair Queries

## Problem Statement
You are given an integer array `nums` of length `n` and an integer array `queries`.

Let `gcdPairs` denote an array obtained by calculating the GCD of all possible pairs `(nums[i], nums[j])`, where `0 <= i < j < n`, and then sorting these values in ascending order.

For each query `queries[i]`, you need to find the element at index `queries[i]` in `gcdPairs`.

Return an integer array `answer`, where `answer[i]` is the value at `gcdPairs[queries[i]]` for each query.

The term `gcd(a, b)` denotes the greatest common divisor of `a` and `b`.

## Examples

### Example 1
```
Input: nums = [2,3,4], queries = [0,2,2]
Output: [1,2,2]
Explanation:
gcdPairs = [gcd(nums[0], nums[1]), gcd(nums[0], nums[2]), gcd(nums[1], nums[2])] = [1, 2, 1].
After sorting in ascending order, gcdPairs = [1, 1, 2].
So, the answer is [gcdPairs[queries[0]], gcdPairs[queries[1]], gcdPairs[queries[2]]] = [1, 2, 2].
```

### Example 2
```
Input: nums = [4,4,2,1], queries = [5,3,1,0]
Output: [4,2,1,1]
Explanation:
gcdPairs sorted in ascending order is [1, 1, 1, 2, 2, 4].
```

### Example 3
```
Input: nums = [2,2], queries = [0,0]
Output: [2,2]
Explanation:
gcdPairs = [2].
```

## Constraints
- `2 <= n == nums.length <= 10^5`
- `1 <= nums[i] <= 5 * 10^4`
- `1 <= queries.length <= 10^5`
- `0 <= queries[i] < n * (n - 1) / 2`

---

## Solution Approach

We cannot explicitly generate all pairs because the number of pairs can be up to ~5e9, which is too large. Instead, we need to count how many pairs have a given GCD value, then answer queries via prefix sums and binary search.

### Steps

1. **Frequency Array**  
   Let `M` be the maximum value in `nums`. Build an array `cnt[1..M]` where `cnt[x]` is the frequency of value `x` in `nums`.

2. **Count Numbers Divisible by `d`**  
   For each `d` from `1` to `M`, compute `multiples[d] = sum_{k>=1} cnt[k*d]`, i.e., the total count of numbers in `nums` that are divisible by `d`.  
   This can be done in O(M log M) by iterating over multiples.

3. **Count Pairs with GCD Divisible by `d`**  
   If we pick any two numbers that are both divisible by `d`, then their GCD is at least `d` (i.e., divisible by `d`). The number of such pairs is:
   ```
   pairsDivisible[d] = C(multiples[d], 2) = multiples[d] * (multiples[d] - 1) / 2
   ```
   This is the total number of pairs whose GCD is a multiple of `d`.

4. **Count Pairs with GCD Exactly `d`**  
   Let `exact[d]` be the number of pairs whose GCD is exactly `d`.  
   We have the relationship:
   ```
   pairsDivisible[d] = sum_{k>=1} exact[k*d]
   ```
   Therefore, we can compute `exact` in descending order:
   ```
   exact[d] = pairsDivisible[d] - sum_{k>=2} exact[k*d]
   ```
   This is essentially the Möbius inversion, but we can compute it by subtracting the contributions of multiples.

5. **Prefix Sum Over GCD Values**  
   Build a prefix array `pref[d] = sum_{i=1..d} exact[i]`.  
   Then `pref[d]` gives the total number of pairs whose GCD is **at most** `d`. Since `pref` is non-decreasing, we can answer each query by finding the smallest `d` such that `pref[d] > queries[i]` (using 0‑based indexing for the sorted list). That `d` is the answer for that query.

6. **Binary Search per Query**  
   For each query index `q` (0‑based), find the smallest `d` with `pref[d] > q`. Because the sorted `gcdPairs` list is indexed from 0, we need the smallest GCD value such that the count of pairs with GCD ≤ that value is greater than `q`.  
   Use binary search (or `Arrays.binarySearch`) over the `pref` array.

### Complexity
- **Time**: O(M log M + Q log M), where `M = max(nums)` ≤ 5e4 and `Q = queries.length` ≤ 1e5.  
- **Space**: O(M) for the various counting arrays.

---

## Java Code

```java
class Solution {
    public int[] gcdValues(int[] nums, long[] queries) {
        int maxVal = 0;
        for (int x : nums) maxVal = Math.max(maxVal, x);

        // Frequency of each value
        int[] cnt = new int[maxVal + 1];
        for (int x : nums) cnt[x]++;

        // multiples[d] = number of elements divisible by d
        long[] multiples = new long[maxVal + 1];
        for (int d = 1; d <= maxVal; d++) {
            long sum = 0;
            for (int m = d; m <= maxVal; m += d) {
                sum += cnt[m];
            }
            multiples[d] = sum;
        }

        // pairsDivisible[d] = number of pairs whose GCD is divisible by d
        long[] pairsDivisible = new long[maxVal + 1];
        for (int d = 1; d <= maxVal; d++) {
            long c = multiples[d];
            pairsDivisible[d] = c * (c - 1) / 2;
        }

        // exact[d] = number of pairs whose GCD is exactly d
        long[] exact = new long[maxVal + 1];
        for (int d = maxVal; d >= 1; d--) {
            long subtract = 0;
            for (int m = 2 * d; m <= maxVal; m += d) {
                subtract += exact[m];
            }
            exact[d] = pairsDivisible[d] - subtract;
        }

        // prefix[d] = total pairs with GCD <= d
        long[] prefix = new long[maxVal + 1];
        for (int d = 1; d <= maxVal; d++) {
            prefix[d] = prefix[d - 1] + exact[d];
        }

        int[] ans = new int[queries.length];
        for (int i = 0; i < queries.length; i++) {
            long q = queries[i];
            int lo = 1, hi = maxVal;
            while (lo < hi) {
                int mid = (lo + hi) / 2;
                if (prefix[mid] > q) {
                    hi = mid;
                } else {
                    lo = mid + 1;
                }
            }
            ans[i] = lo;
        }

        return ans;
    }
}
