# Lcode - LeetCode Daily Solutions

A personal repository for storing solutions to LeetCode (Lcode) coding challenges, solved on a daily basis. This repository serves as a record of algorithmic progress and a reference for various problem-solving techniques.

## 🚀 Overview

The solutions in this repository are primarily implemented in **Java** and **Python**, covering a wide range of topics including dynamic programming, graph theory, data structures, and more.

## 📁 Repository Structure

The project is organized chronologically to track daily progress:

- **`2025-Files/`**: Contains an extensive archive of solutions from the year 2025 (over 180 solutions).
- **`April26/`**: Current active directory for solutions solved in April 2026.
- **Root Directory**: Contains project configuration and a historical log of solutions.

### Naming Convention

Files are named based on the date the problem was solved:
- Format: `DD_MM_YYYY` or `DD-MM-YYYY`
- Example: `01_05_2025` (May 1st, 2025) or `03-04-2026.txt`

## 🛠️ Languages Used

- **Java**: used for most of the 2025 solutions.
- **Python**: used for recent 2026 solutions.
