
# 3534. Path Existence Queries in a Graph II (Hard)

## Problem Description

You are given an integer `n` representing the number of nodes in a graph, labeled from `0` to `n - 1`.

You are also given an integer array `nums` of length `n` and an integer `maxDiff`.

An undirected edge exists between nodes `i` and `j` if the absolute difference between `nums[i]` and `nums[j]` is at most `maxDiff` (i.e., `|nums[i] - nums[j]| <= maxDiff`).

You are also given a 2D integer array `queries`. For each `queries[i] = [ui, vi]`, find the **minimum distance** between nodes `ui` and `vi`. If no path exists between the two nodes, return `-1` for that query.

Return an array `answer`, where `answer[i]` is the result of the `i`th query.

**Note:** The edges between the nodes are unweighted.

## Examples

### Example 1
```
Input: n = 5, nums = [1,8,3,4,2], maxDiff = 3, queries = [[0,3],[2,4]]
Output: [1,1]
```
**Explanation:**  
The graph has edges where the absolute difference ≤ 3.  
- Query [0,3]: 0 → 3 directly, distance = 1.  
- Query [2,4]: 2 → 4 directly, distance = 1.

### Example 2
```
Input: n = 5, nums = [5,3,1,9,10], maxDiff = 2, queries = [[0,1],[0,2],[2,3],[4,3]]
Output: [1,2,-1,1]
```
**Explanation:**  
- [0,1]: 0 → 1 (diff = 2) distance 1.  
- [0,2]: 0 → 1 → 2 (5→3→1) distance 2.  
- [2,3]: 2 (value 1) and 3 (value 9) diff 8 > 2, no path → -1.  
- [4,3]: 4 (10) → 3 (9) diff 1, distance 1.

### Example 3
```
Input: n = 3, nums = [3,6,1], maxDiff = 1, queries = [[0,0],[0,1],[1,2]]
Output: [0,-1,-1]
```
**Explanation:**  
No edges exist because all differences exceed 1.  
- [0,0] same node → distance 0.

## Constraints
- `1 <= n == nums.length <= 10^5`
- `0 <= nums[i] <= 10^5`
- `0 <= maxDiff <= 10^5`
- `1 <= queries.length <= 10^5`
- `queries[i] == [ui, vi]`
- `0 <= ui, vi < n`

## Solution Approach

The graph is defined by a threshold on absolute difference. We sort nodes by their `nums` value. For each node, the set of its neighbors forms a contiguous interval in the sorted order (all indices whose values differ by ≤ `maxDiff`). Thus, from any node, we can jump to the leftmost or rightmost index reachable in one step.

The problem reduces to finding the shortest number of jumps between two indices in the sorted array, where each jump can move to any position within a sliding window (interval). This can be solved with **binary lifting** on the left and right jump functions:

- `left[i]` = smallest index reachable from index `i` (including itself) in one jump.
- `right[i]` = largest index reachable from index `i` in one jump.

Then we precompute binary lifting tables for both directions. For a query, we convert original node IDs to sorted indices and check if we can move left or right to reach the target using the minimal number of jumps.

**Time Complexity:** O((n + q) log n)  
**Space Complexity:** O(n log n)

## Java Solution

```java
import java.util.Arrays;

class Solution {
    public int[] pathExistenceQueries(int n, int[] nums, int maxDiff, int[][] queries) {
        // Sort nodes by nums value
        Integer[] order = new Integer[n];
        for (int i = 0; i < n; i++) order[i] = i;
        Arrays.sort(order, (a, b) -> Integer.compare(nums[a], nums[b]));

        int[] sorted = new int[n];
        int[] pos = new int[n];
        for (int i = 0; i < n; i++) {
            sorted[i] = nums[order[i]];
            pos[order[i]] = i;
        }

        // left[i]  = leftmost index reachable from i in one edge
        // right[i] = rightmost index reachable from i in one edge
        int[] left = new int[n];
        int[] right = new int[n];

        for (int i = 0; i < n; i++) {
            int target = sorted[i] - maxDiff;
            int lo = 0, hi = n - 1;
            while (lo < hi) {
                int mid = (lo + hi) / 2;
                if (sorted[mid] >= target) hi = mid;
                else lo = mid + 1;
            }
            left[i] = lo;

            target = sorted[i] + maxDiff;
            lo = 0;
            hi = n - 1;
            while (lo < hi) {
                int mid = (lo + hi + 1) / 2;
                if (sorted[mid] <= target) lo = mid;
                else hi = mid - 1;
            }
            right[i] = lo;
        }

        // Binary lifting on left/right jump functions
        int LOG = 1;
        while ((1 << LOG) <= n) LOG++;

        int[][] upLeft = new int[LOG][n];
        int[][] upRight = new int[LOG][n];

        for (int i = 0; i < n; i++) {
            upLeft[0][i] = left[i];
            upRight[0][i] = right[i];
        }

        for (int k = 1; k < LOG; k++) {
            for (int i = 0; i < n; i++) {
                upLeft[k][i] = upLeft[k - 1][upLeft[k - 1][i]];
                upRight[k][i] = upRight[k - 1][upRight[k - 1][i]];
            }
        }

        int[] ans = new int[queries.length];

        for (int qi = 0; qi < queries.length; qi++) {
            int u = queries[qi][0];
            int v = queries[qi][1];

            int pu = pos[u];
            int pv = pos[v];

            if (pu == pv) {
                ans[qi] = 0;
                continue;
            }

            if (pu > pv) {
                // Target is to the left: use left jumps
                int cur = pu;
                int steps = 0;

                for (int k = LOG - 1; k >= 0; k--) {
                    int nxt = upLeft[k][cur];
                    if (nxt > pv) {
                        cur = nxt;
                        steps += 1 << k;
                    }
                }

                int nxt = upLeft[0][cur];
                if (nxt <= pv) ans[qi] = steps + 1;
                else ans[qi] = -1;
            } else {
                // Target is to the right: use right jumps
                int cur = pu;
                int steps = 0;

                for (int k = LOG - 1; k >= 0; k--) {
                    int nxt = upRight[k][cur];
                    if (nxt < pv) {
                        cur = nxt;
                        steps += 1 << k;
                    }
                }

                int nxt = upRight[0][cur];
                if (nxt >= pv) ans[qi] = steps + 1;
                else ans[qi] = -1;
            }
        }

        return ans;
    }
}
