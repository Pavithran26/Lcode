# 1081. Smallest Subsequence of Distinct Characters

**Medium** | **Topics:** String, Stack, Greedy

## Problem Statement

Given a string `s`, return the **lexicographically smallest subsequence** of `s` that contains all the distinct characters of `s` exactly once.

### Example 1:
```
Input: s = "bcabc"
Output: "abc"
```

### Example 2:
```
Input: s = "cbacdcbc"
Output: "acdb"
```

### Constraints:
- `1 <= s.length <= 1000`
- `s` consists of lowercase English letters.

> **Note:** This question is the same as LeetCode 316: [Remove Duplicate Letters](https://leetcode.com/problems/remove-duplicate-letters/)

---

## Approach: Monotonic Stack with Greedy

We need the smallest lexicographical subsequence while keeping **each distinct character exactly once**.  

### Key Idea:
1. **Track last occurrence** of each character – this tells us if a character will appear again later.
2. Use a **stack** to build the result.
3. For each character `c`:
   - If `c` is already used, skip it.
   - While the stack is not empty, the top of the stack is **greater** than `c`, and that top character appears **again later** in the string, we pop it (because we can pick it later, and using `c` now gives a smaller lexicographical order).
   - Push `c` onto the stack and mark it as used.

### Complexity:
- **Time:** O(n) – each character is pushed and popped at most once.
- **Space:** O(1) – stack size is at most 26 (number of lowercase letters).

---

## Java Solution

```java
class Solution {
    public String smallestSubsequence(String s) {
        int n = s.length();
        
        // last occurrence index of each character
        int[] last = new int[26];
        for (int i = 0; i < n; i++) {
            last[s.charAt(i) - 'a'] = i;
        }
        
        boolean[] seen = new boolean[26];
        char[] stack = new char[n];
        int top = -1;
        
        for (int i = 0; i < n; i++) {
            char c = s.charAt(i);
            int idx = c - 'a';
            
            // Each character must appear exactly once
            if (seen[idx]) continue;
            
            // Remove larger characters if they appear later and current char is smaller
            while (top >= 0 && stack[top] > c && last[stack[top] - 'a'] > i) {
                seen[stack[top] - 'a'] = false;
                top--;
            }
            
            stack[++top] = c;
            seen[idx] = true;
        }
        
        return new String(stack, 0, top + 1);
    }
}
```
