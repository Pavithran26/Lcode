# LeetCode 3754 - Concatenate Non-Zero Digits and Multiply by Sum I

## 📌 Problem Statement

You are given an integer `n`.

Form a new integer `x` by concatenating all the non-zero digits of `n` in their original order. If there are no non-zero digits, `x = 0`.

Let `sum` be the sum of digits in `x`.

Return an integer representing the value of `x * sum`.

---

## 🎯 Objective

Remove all `0` digits from the given number, create a new number using the remaining digits in the same order, calculate the sum of those digits, and return:

```text
Answer = x × sum
```

---

## 🧪 Examples

### Example 1

**Input**

```text
n = 10203004
```

**Output**

```text
12340
```

**Explanation**

* Non-zero digits = `1, 2, 3, 4`
* New number (`x`) = `1234`
* Sum of digits = `1 + 2 + 3 + 4 = 10`
* Answer = `1234 × 10 = 12340`

---

### Example 2

**Input**

```text
n = 1000
```

**Output**

```text
1
```

**Explanation**

* Non-zero digits = `1`
* New number (`x`) = `1`
* Sum = `1`
* Answer = `1 × 1 = 1`

---

## 🔍 Constraints

```text
0 <= n <= 10^9
```

---

# 💡 Intuition

The problem asks us to ignore every `0`.

While reading each digit:

* If the digit is `0`, skip it.
* Otherwise:

  * Add it to the new number.
  * Add its value to the digit sum.

After processing all digits:

* If there were no non-zero digits, return `0`.
* Otherwise return:

```text
x × digit_sum
```

---

# 📝 Approach

### Step 1

Convert the integer into a string so each digit can be processed easily.

### Step 2

Create:

```python
x = ""
digit_sum = 0
```

### Step 3

Traverse every digit.

If digit is not `'0'`

* Append it to `x`
* Add it to `digit_sum`

### Step 4

If `x` is empty

Return

```python
0
```

### Step 5

Otherwise

```python
return int(x) * digit_sum
```

# ✅ Python Solution

```python
class Solution:
    def sumAndMultiply(self, n: int) -> int:
        x = ""
        digit_sum = 0

        for ch in str(n):
            if ch != '0':
                x += ch
                digit_sum += int(ch)

        if x == "":
            return 0

        return int(x) * digit_sum
```

# 🎯 Key Takeaways

* Convert the number to a string for easy digit traversal.
* Ignore all `0` digits.
* Build the new number and calculate the digit sum in a single pass.
* Remember to handle the edge case where all digits are `0`.
